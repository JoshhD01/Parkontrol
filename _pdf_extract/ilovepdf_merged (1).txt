●  Backend  con  base  de  datos  relacional  ○  Oracle  XE  con  modelos  normalizados.  ○  API  REST  para  consumo  de  datos  (FastAPI).  ●  Interfaz  para  operadores  y  administradores  ○  Panel  simple  para  registrar  entradas/salidas.  ○  Dashboard  administrativo  con  métricas  y  reportes.  ●  Soporte  multi  empresa  (multi-tenant)  ○  Cada  cliente  (centro  comercial,  hospital,  universidad)  tendrá  sus  propios  datos
